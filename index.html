Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML5

    HTML5 (HyperText Markup Languaje, versión 5) es la quinta revisión importante del lenguaje básico de la Word Wide Web. HTML.

    Usa la extensión .html

  • CSS

    Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets)es un lenguaje usando par definir y crear la presentacion de un documento estructurado en HTML.

    Usa la extensión css

  • JS

    JavaScript(abreviado comunmente "JS"") es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en el interfaz de usuario y páginas web dinámicas.

    Usa la extensión .js
